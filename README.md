This project is a Python-based Statistical Critical Value Calculator developed to simplify hypothesis testing by eliminating the need to manually search through statistical tables. It supports Z, t, Chi-Square, and F distributions, allowing users to calculate critical values quickly through a simple graphical interface built with Tkinter. The application uses SciPy to ensure accurate calculations and also provides the hypothesis testing decision based on the user's inputs. I built this project to strengthen my Python skills while creating a practical tool that can save time for students, researchers, and anyone working with statistical analysis.
